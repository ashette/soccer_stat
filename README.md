# soccer-stat

## Project setup
```
npm install
```

Then rename `.env.example` file to `.env` and replace `YOUR_TOKEN` with your API-token
```
VUE_APP_SOCCER_API_KEY=YOUR_TOKEN
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

Enjoy!
